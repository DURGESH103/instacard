## Getting Started

1. Clone the repository
```bash
git clone https://github.com/DURGESH103/instacard.git
cd instacard
```

2. Install dependencies
```bash
npm install
```

3. Start development server
```bash
npm start
```

4. Build for production
```bash
npm run build
```
